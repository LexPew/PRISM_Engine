## Getting Started

### Prerequisites
Make sure you have the following installed:
- **C++ compiler** (C++17 or newer recommended)
- **CMake** (>= 3.28)

### Build Instructions
```bash
# Clone the repository
git clone [https://github.com/your-username/your-repo.git](https://github.com/LexPew/OpenGLEngine.git)
cd your-repo

# Run CMake
cmake -B build

# Build the project
cmake --build build
